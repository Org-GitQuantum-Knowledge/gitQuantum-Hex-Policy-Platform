API Reference

PolicyVM

The core virtual machine for executing compiled policy bytecode.

Constructor

constructor(bytecode: any[])

Creates a new PolicyVM instance with compiled bytecode.

Methods

execute(ctx: Context): Decision

Executes the bytecode against the provided context and returns a decision.

const vm = new PolicyVM(bytecode);
const decision = vm.execute(context);
// Returns: 'ALLOW' | 'DENY'

Compiler

Transforms policy rules into optimized bytecode.

Functions

compileToBytecode(rules: any[]): any[]

Compiles an array of policy rules into bytecode format.

import { compileToBytecode } from '@hex/core';

const rules = [
  { attribute: 'role', element: 'document', perm: 'read' }
];
const bytecode = compileToBytecode(rules);

Type Definitions

Context

interface Context {

}

Decision

type Decision = 'ALLOW' | 'DENY';

Benchmark API

The benchmark suite provides performance analysis tools.

BenchmarkRunner

import { BenchmarkRunner } from '@hex/benchmark';

const runner = new BenchmarkRunner({
  iterations: 100000,
  ruleCount: 10,
  warmup: true
});

const result = runner.run();
// Returns: BenchmarkResult

BenchmarkResult

interface BenchmarkResult {
  hexTime: number;      // HEX VM execution time in ms
  opaTime: number;      // OPA simulator execution time in ms
  speedup: number;      // Speedup factor (opaTime / hexTime)
  iterations: number;   // Number of iterations
  timestamp: Date;      // When the benchmark was run
}
